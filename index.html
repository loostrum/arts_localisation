ARTS Localisation¶

This project contains all necessary tools to localise transient bursts discovered with the Apertif Radio Transient System (ARTS). The localisation uses a chi squared method, comparing the S/N of a detected burst across beams to a model of the telescope response. The beam model includes a simulation of the ARTS tied-array beamformer and the generation of synthesised beams (SBs). The beam model and localisation method are described in more detail in my PhD thesis: Fast Radio Bursts with Apertif.

Localisation overview¶

In short, the localistion consists of the following steps:

  1. Note in which CBs the FRB was detected.

  2. Save a snippet for the detection CBs, as well as for all surrounding CBs.

  3. Create a configuration file with the settings to use to S/N determination

  4. Run arts_calc_snr

  5. Add localisation settings to the configuration file, using coarse resolution

  6. Run arts_run_localisation

  7. Repeat the previous step with different settings until the entire uncertainty region is covered

  8. Re-run the localisation at high resolution
